# bi-Business Intelligence

AdditionalBill - 加帐
Contract - 合同
DeptCstCtrl - 部门成本控制
MajorDevCst - 主要设备费用
MajorMatCst - 主材费
MajorMatQty - 主材用量
Project - 单船总信息
ProjectCost - 单船成本
ProjectCstCtrl - 单船成本控制
ProjectInProcess - 在建船舶
ProjectManhour - 单船工时
ProjectSchedule - 船舶大节点
